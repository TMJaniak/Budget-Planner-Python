# Budget-Planner-Python
This program was the final project in my IT-4401 class at The University Of Missouri. It is used to determine a budget for a user based on the expenses for selected months.

## Features
- Reads expense data from multiple monthly text files
- Aggregates expenses by category
- Calculates total income, expenses, and remaining balance
- Automatically allocates sinking funds for non-regular expenses
- Allows dynamic month selection by the user

## Technologies Used
- Python 3
- File I/O

## How to Run
1. Clone the repository
2. Navigate to the project directory
3. Ensure Python 3 is installed
4. Run the program
5. Follow the on-screen prompts
